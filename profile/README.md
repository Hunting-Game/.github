## About

Hunters track runners on a real-time map with GPS precision. Runners evade using only a compass pointing toward hunters. Use powerful special abilities like invisibility, blinding opponents, or revealing enemy locations to gain the upper hand. Play anywhere outdoors with unlimited friends.
